# Task 1 – Rating Prediction via Prompting

This task evaluates different prompting strategies for predicting Yelp review ratings.

## Files
- rating_prediction_prompting.ipynb – main notebook
- data/yelp_reviews_sample.csv – sampled dataset (200 rows)
- prompts/ – prompt versions used in experiments

## Prompt Versions
- V1: Zero-shot
- V2: Few-shot
- V3: Rubric-based

## Evaluation Metrics
- Accuracy
- JSON validity
- Consistency
